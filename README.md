# Address-Book

## Deskripsi
Aplikasi sederhana untuk menyimpan dan mengelola daftar kontak dengan fitur tambah, hapus, cari, dan update.

## Fitur
- Lihat daftar kontak
- Tambah kontak baru
- Hapus kontak
- Cari kontak

## Alur Aplikasi
1. Tampilkan menu utama ke pengguna
2. Pengguna memilih fitur
3. Jalankan fitur sesuai pilihan
4. Kembali ke menu utama atau keluar aplikasi
